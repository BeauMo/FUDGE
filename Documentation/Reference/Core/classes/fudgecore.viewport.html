Class Viewport

Controls the rendering of a branch, using the given ComponentCamera, and the propagation of the rendered image from the offscreen renderbuffer to the target canvas through a series of Framing objects. The stages involved are in order of rendering [[RenderManager]].viewport -> Viewport.source -> Viewport.destination -> DOM-Canvas -> Client(CSS)
